# cctweakedprograms

CC:Tweaked (ComputerCraft) Lua programs.

## Install

In-game, at any computer's Lua prompt:

```
wget run https://raw.githubusercontent.com/XxXRAYQUAZAXxX/cctweakedprograms/main/install.lua
```

## Contents

- `remote_agent.lua` / `remote_controller.lua` — in-game remote computer control over rednet (like Cloud Catcher, but no internet needed). Run `remote_agent` on any computer you want reachable, `remote_controller` on the computer you want to control from.
